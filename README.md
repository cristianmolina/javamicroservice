# javamicroservice

# User Microservice

## Requirements
- Java 8
- Gradle 7.4

## Building
```bash
./gradlew build
```

## Running
```bash
./gradlew bootRun
```

## Endpoints
- POST /users/sign-up
- GET /users/login
